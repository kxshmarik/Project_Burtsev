# SSL Checker

Инструмент для проверки SSL-сертификатов.

## Что умеет

- Проверять срок действия SSL-сертификатов
- Смотреть цепочку сертификатов (при необходимости)
- Проверять несколько сайтов сразу
- Выдавать краткие рекомендации по обновлению сертификатов

## Установка

1. Установить Python 3.7 или новее
2. Установить зависимости:

```bash
pip install cryptography
